# OmegaTK RAG Coding Assistant

A retrieval-augmented generation (RAG) coding assistant for the
[OmegaTK Python toolkit](https://docs.eyesopen.com/toolkits/python/omegatk/)
by OpenEye Scientific Software (Cadence Molecular Sciences).

> **Status:** 🚧 In development

## Overview

This assistant answers questions about the OmegaTK API and generates
executable Python code by retrieving relevant documentation at query time.

## Planned Features

- FAISS vector index over live OmegaTK documentation
- Gemini 2.5 Flash LLM with RAG context
- Streamlit web UI with chat history
- Automated test suite
